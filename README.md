# 🎮 Consoles e Jogos JG

Projeto desenvolvido em **C# com ASP.NET Core Razor Pages**, simulando uma loja virtual de videogames, consoles e acessórios.  
O sistema tem como objetivo praticar conceitos de desenvolvimento web, organização de páginas, integração entre front-end e back-end e manipulação de dados utilizando a plataforma .NET.

---

## 📌 Funcionalidades

- Cadastro de produtos
- Listagem de jogos e consoles
- Página de detalhes do produto
- Sistema de categorias
- Carrinho de compras (esboço)
- Layout responsivo
- Navegação entre páginas usando Razor Pages

---

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/consoles-e-jogos-jg.git
```

2. Acesse a pasta do projeto:

```bash
cd consoles-e-jogos-jg
```

3. Execute o projeto:

```bash
dotnet run
```

4. Abra o navegador no endereço:

```bash
https://localhost:5001
```
